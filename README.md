## 功能特性

- ✅ OAuth 2.0 认证支持
- ✅ 完整的任务管理功能
- ✅ 项目管理功能
- ✅ TypeScript 支持
- ✅ 安全的 API 访问

## 快速开始

```bash
# 克隆项目
git clone https://github.com/vcve/dida365-mcp-server-enhanced.git

# 安装依赖
cd dida365-mcp-server-enhanced
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填入您的 API 密钥

# 构建项目
npm run build

# 运行服务器
node dist/index.js
```
